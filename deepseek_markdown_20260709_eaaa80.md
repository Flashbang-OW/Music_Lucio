# 🐸 Lúcio Music Player

Um player de música temático do Lúcio de Overwatch. Porque vamos aumentar o som!

## 🎵 Funcionalidades

- Player de música completo
- Playlist temática do Lúcio
- Design inspirado no herói
- Animações e efeitos visuais
- Preparado para integração com Supabase

## 🚀 Como usar

1. Clona o repositório
2. Abre o `index.html` no navegador
3. Curte o som!

## 🎨 Tecnologias

- HTML5
- CSS3
- JavaScript puro
- Supabase (futuro)
